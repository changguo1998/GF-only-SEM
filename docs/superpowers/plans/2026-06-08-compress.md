## Task 4 — `ChunkingStrategy.h`

**File:** `compress/include/gf/ChunkingStrategy.h`

**Target strain shape:** `[n_snapshot, n_elem_local, NGLL, NGLL, NGLL, 6]`.

**Chunk rule:**

- Time: `1`.
- Element: `min(chunk_size, n_elem_local)`; default `64`.
- Spatial/component dims: full `NGLL, NGLL, NGLL, 6`.

**Work:**

- [x] Add `ChunkingConfig`.
- [x] Add `make_strain_chunks(...)`.
- [x] Validate positive dimensions.
- [x] Commit.
