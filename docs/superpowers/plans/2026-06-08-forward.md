# Forward Solver — Green's Function SEM Core Implementation Plan

> Use `subagent-driven-development` or `executing-plans` to run tasks. Check boxes track progress.

**Goal:** Build `libgf` and MPI executable `gf_solver` for elastic SEM forward modeling.

**Design:** `docs/superpowers/design/forward.md`.

## Run Config

Run three independent force directions per source: `x`, `y`, `z`. Postprocess stacks 6 strain components from those runs into the Green tensor.
