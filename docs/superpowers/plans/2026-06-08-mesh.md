## Design Rules

| Rule | Example |
|------|---------|
| `X2Y` naming for relations | `edge_to_vertex`, `cell_to_surface` |
| 1-based indexing, 0 = null | element IDs, vertex IDs, surface IDs |
| Sign = direction (signed int) | `+edge_id` = positive traversal, `-edge_id` = reverse |

---

## Topology Hierarchy

```
vertex → edge → surface → cell
         (2 vertices)   (4 edges, quad)   (6 surfaces, hex)
```

---

## HDF5 Output Schema — `mesh.h5`

Converter does NO computation. It reads GMSH v4.1 and writes topology only.

```
mesh.h5
└── /topology/
    ├── n_vertex         : attr int64
    ├── n_edge           : attr int64
    ├── n_surface        : attr int64
    ├── n_cell           : attr int64
    │
    ├── vertex_to_coord  : float64[n_vertex, 3]           — (x, y, z), vertex ID = 1-based row index
    │
    ├── edge_to_vertex   : int64[n_edge, 2]               — (+v1, +v2), v1 < v2
    │                                                       positive direction = v1 → v2
    │
    ├── surface_to_edge  : int64[n_surface, 4]            — (±edge_id, ...) CCW loop
    │                                                       +edge = traverse in positive edge direction
    │                                                       -edge = traverse in reverse
    │
    └── cell_to_surface  : int64[n_cell, 6]               — (±surface_id, ...)
                                                              +surface = outward normal
                                                              -surface = inward normal
```

Materials, boundaries, GLL node positions, geometric quantities, and partition info are NOT in `mesh.h5`. They are all computed by the preprocessor and stored in `partition_{r}.h5`.

## Optional: `mesh_auxiliary.h5`

CSR adjacency relations for validation and algorithm acceleration:

```
mesh_auxiliary.h5
└── /auxiliary/
    ├── surface_to_cell         : int64[n_surface, 2]
    ├── vertex_to_edge          : CSR (indptr[n_vertex+1], indices)
    ├── vertex_to_surface       : CSR
    ├── vertex_to_cell          : CSR
    ├── edge_to_surface         : CSR (indptr[n_edge+1], indices)
    └── edge_to_cell            : CSR
```

Generated by the converter (or preprocessor) for convenience.

```python
#!/usr/bin/env python3
"""Convert GMSH v4.1 .msh to internal HDF5 topology format.

Reads GMSH mesh, extracts topology only (vertices, edges, surfaces, cells),
writes mesh.h5 with /topology/ group. 1-based indexing, signed direction.

Usage:
    python tools/gmsh_to_hdf5.py input.msh -o mesh.h5 [--aux mesh_auxiliary.h5]
"""

import argparse
import h5py
import meshio
import numpy as np


def main():
    parser = argparse.ArgumentParser(description="GMSH → HDF5 topology converter")
    parser.add_argument("input", help="Input GMSH .msh file (v4.1)")
    parser.add_argument("-o", "--output", required=True, help="Output mesh.h5")
    parser.add_argument("--aux", help="Optional auxiliary CSR file")
    args = parser.parse_args()

    # Read GMSH mesh
    mesh = meshio.read(args.input)

    # Extract topology
    topology = extract_topology(mesh)

    # Write mesh.h5
    write_topology(args.output, topology)

    # Optional auxiliary
    if args.aux:
        write_auxiliary(args.aux, topology)


if __name__ == "__main__":
    main()
```

```python
def extract_topology(mesh):
    """Extract vertices, edges, surfaces, cells from GMSH mesh.
    
    Returns dict with:
      - vertex_to_coord: (n_vertex, 3) float64 — 1-based vertex coords
      - edge_to_vertex: (n_edge, 2) int64 — (+v1, +v2), v1 < v2
      - surface_to_edge: (n_surface, 4) int64 — (±edge_id, ...) CCW loop
      - cell_to_surface: (n_cell, 6) int64 — (±surface_id, ...) outward normal
    """
    # 1. Extract vertices (1-based)
    n_vertex = mesh.points.shape[0]
    vertex_to_coord = mesh.points.astype(np.float64)
    
    # 2. Build edge table from GMSH cell connectivity
    # GMSH hex order → infer 12 edges, deduplicate
    
    # 3. Build surface table from GMSH cell faces
    # GMSH hex order → 6 quad faces per cell
    
    # 4. Build cell_to_surface mapping
    # ±surface_id depending on outward/inward normal
    
    return {
        "vertex_to_coord": vertex_to_coord,
        "edge_to_vertex": edge_to_vertex,
        "surface_to_edge": surface_to_edge,
        "cell_to_surface": cell_to_surface,
    }
```

Key implementation details:
- **Edge deduplication**: Sort vertex pair (v_a, v_b) for uniqueness, store +v1, +v2 with v1 < v2 for canonical orientation
- **Surface orientation**: Each hex face has 4 edges in CCW order when viewed from outside. Sign each edge based on whether traversal matches the canonical edge direction (+v1→v2)
- **Cell surface signs**: +surface when the surface normal points outward from the cell, -surface otherwise

```python
def write_topology(path, topology):
    """Write mesh.h5 with /topology/ group. 1-based indexing, signed int."""
    with h5py.File(path, "w") as f:
        topo = f.create_group("topology")
        
        v2c = topology["vertex_to_coord"]
        e2v = topology["edge_to_vertex"]
        s2e = topology["surface_to_edge"]
        c2s = topology["cell_to_surface"]
        
        n_vertex = v2c.shape[0]
        n_edge   = e2v.shape[0]
        n_surface = s2e.shape[0]
        n_cell   = c2s.shape[0]
        
        # Write datasets
        topo.create_dataset("vertex_to_coord", data=v2c, dtype="float64")
        topo.create_dataset("edge_to_vertex",  data=e2v, dtype="int64")
        topo.create_dataset("surface_to_edge", data=s2e, dtype="int64")
        topo.create_dataset("cell_to_surface", data=c2s, dtype="int64")
        
        # Write counts as attributes
        for name, count in [("n_vertex", n_vertex), ("n_edge", n_edge),
                            ("n_surface", n_surface), ("n_cell", n_cell)]:
            topo.attrs[name] = np.int64(count)
```
